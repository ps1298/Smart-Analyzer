# Smart Analyzer

Smart Analyzer is a desktop-based analytics application built using Python.

## Features
- CSV/Excel Analysis
- Dynamic Reports
- Chart Visualization
- Export to Excel/CSV
- Dark/Light Mode
- User-friendly GUI

## Technologies
- Python
- Tkinter
- Pandas
- Matplotlib

## Project Type
AI-assisted Prompt Engineering with Python project.
